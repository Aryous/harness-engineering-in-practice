# Harness Engineering 实践教程

让 AI Agent 可控产出的工程方法。

用同一个模型（Claude Opus 4.6）、同一套工具（Claude Code）、同一个领域（简历编辑器），两个项目做对照实验。唯一变量是有没有 Harness——约束、文档、机械验证构成的 Agent 运行环境。
